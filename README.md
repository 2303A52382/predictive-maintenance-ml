# Predictive Maintenance using Machine Learning

## 📌 Project Overview
This project predicts machine failure using supervised machine learning models.
The goal is to help industries prevent unexpected breakdowns and reduce maintenance costs.

## 📊 Dataset
AI4I 2020 Predictive Maintenance Dataset.

Features include:
- Air temperature
- Process temperature
- Rotational speed
- Torque
- Tool wear
- Machine failure (Target variable)

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## 🤖 Models Implemented
1. Logistic Regression
2. Random Forest (Hyperparameter Tuned)

## 📈 Model Performance
- Accuracy: 99%
- Confusion Matrix visualization included
- Class imbalance handled

## 🚀 How to Run This Project
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   jupyter notebook predictive_maintenance.ipynb
